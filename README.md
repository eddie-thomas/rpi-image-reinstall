# rpi-image-reinstall

### Install `rpi-imager`

1. run `sudo pacman -Syu`
2. run `git clone https://aur.archlinux.org/rpi-imager.git`
3. run `cd rpi-imager`
4. run `makepkg -si`

Run the `rpi-imager` command to launch the OS writer. Use the micro-SD USB reader/writer and pick an OS.
