# rpi-image-reinstall

### Install `rpi-imager`

1. run `sudo pacman -Syu`
2. run `git clone https://aur.archlinux.org/packages/rpi-imager`
3. run `cd rpi-imager`
4. run `makepkg -si`

Documentation explaining what to do when reinstalling the OS (operating system) for the Raspberry Pi on Arch-based system
